# Succs-Essentials-Extemded
### Requires [Curios API](https://modrinth.com/mod/curios) and [TerraBlender](https://modrinth.com/mod/terrablender) to run

### **Succ's Essentials: Extended**

**Succ's Essentials Extened** is a Minecraft mod that adds onto [Succ's Essentials](https://modrinth.com/mod/succs-essentials) with features i felt didn't fit the main mods theme.
